# ♻️ Smart Waste Management System

A modern waste management web application built using Laravel and Tailwind CSS that helps users schedule recyclable waste pickups and promotes sustainable living.

---

## 🚀 Features

- User Authentication
- Waste Pickup Scheduling
- Recycling Awareness Section
- Responsive Modern UI
- Eco-Friendly Design
- Tailwind CSS Styling

---

## 🛠️ Tech Stack

- Laravel
- PHP
- Tailwind CSS
- MySQL
- Vite

---

## ⚙️ Installation

```bash
git clone https://github.com/kundan9199/waste-management-system.git

cd waste-management-system

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

npm run dev

php artisan serve
```

---

## 🌱 Purpose

This project was created to encourage proper waste disposal and sustainable recycling practices through a simple and user-friendly platform.
